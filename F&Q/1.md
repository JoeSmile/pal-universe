## 一、新手入门基础FAQ
### 1. 1.0正式版相比抢先测试版，核心更新有哪些？
1. 地图规模翻倍，新增**天阳乡浮空岛、世界树秘境**两大终局区域，新增冰封山谷、深层熔岩垂直地形、禁猎异变区；
2. 新增72只帕鲁（47全新+25变异亚种），总图鉴287只，重做全部帕鲁数值平衡；
3. 全新养成：**辉石觉醒、突变繁育**两大终局系统；
4. 基建全部重做：古代文明自动化一体机，替代旧版分散工作台/采集场；
5. 主线完整结局，高塔头目新增战斗机制，支持跨平台联机，最多32人专用服务器；
6. 捕捉奖励门槛从12只降至5只，浓缩升星所需帕鲁从116只砍到48只，大幅降低收集压力。

### 2. 旧抢先版存档能直接导入1.0吗？
可以继承角色等级、帕鲁、物资，但**主线/支线任务进度重置**，旧随机帕鲁分布失效；官方强烈建议新开存档体验完整新内容，旧MOD会导致存档崩溃、炸档。

### 3. 开局属性点怎么加最优？
开荒优先级：**负重＞耐力＞生命值＞攻击力**
1. 负重点到1000、耐力200足够日常跑图采集；
2. 生命值拉高提升容错，1.0帕鲁伤害大幅上调，生存优先；
3. 后期再补攻击、捕捉效率词条，不用前期浪费点数。

### 4. 帕鲁终端、队伍、据点盒子有什么区别？
1. **队伍帕鲁**：随身携带，战斗、乘骑、主动采集；
2. **据点帕鲁**：放在基地自动打工、生产物资；
3. **盒子储存**：闲置帕鲁存放，自动回血、恢复 sanity，无数量上限，仅受终端格子限制。

### 5. 帕鲁疯狂罢工、不干活是什么原因？
1. Sanity（理智值）归零，工作速度暴跌，必须放饲料箱+加温/降温设施；
2. 工作适应性等级不够，1.0专精机制削弱混岗全能帕鲁；
3. 工作台无对应原料、输出箱子满了，生产线卡死；
4. 基地超出建造范围，帕鲁无法抵达工位。

### 6. 怎么快速提升捕捉成功率？
1. 打到残血，施加麻痹、冰冻控制debuff；
2. 使用高阶帕鲁球（1.0新增两种顶级球）；
3. 同种类抓满5只解锁永久捕捉加成雕像；
4. 夜晚、雨天捕捉自带概率增益。

## 二、地图探索&区域热门问题
### 1. 天阳乡浮空岛怎么上去？
必须解锁**高级飞行帕鲁（云海巨鲸/电气狮）** 或翼装滑翔，地面无法徒步抵达；产出专属烈阳金属、空中限定帕鲁（拔刀鱼、电气狮）。

### 2. 世界树秘境进入条件是什么？
1. 通关全部高塔主线，解锁终局剧情；
2. 携带辉石解锁三层守卫Boss；
3. 唯一产出觉醒材料辉石、帕鲁树晶矿，概率刷新传说变异帕鲁。

### 3. 禁猎异变区有什么特殊机制？
区域环境动态变化，随机刷新高稀有帕鲁、专属异变首领（霜蚀巨蜥、熔核守卫），击败掉落**异变共鸣石**，用于帕鲁突变繁育；区域内禁止使用部分强力捕捉球。

### 4. 云海巨鲸有什么独有作用？
1. 超大多人飞行坐骑；
2. **唯一能无障碍进入世界树深层区域**的坐骑，其他飞宠会被地形阻挡。

### 5. 1.0隐藏洞窟两种封堵怎么区分？
1. **整块无缝岩壁封闭**：限时轮换地下城副本，固定冷却200分钟刷新，只能等游戏时间；刀剑、镐子无法破坏；
2. **零散碎石堆挡门**：隐藏支线洞窟，可用采矿帕鲁、投掷野怪、爆炸道具击碎，内部藏图纸、稀有物资。

### 6. 限时山洞副本石头挡住进不去怎么办？
1. 单机：存档退出，等待游戏内3小时左右自动刷新岩石消失；可修改本地系统时间加速；
2. 联机服务器：全服统一冷却，无法单人跳时间；
3. 冷却期间洞口无交互提示，岩石完整无缝无碎石。

## 三、Boss、高塔、副本高频问题
### 1. 高塔头目刷新机制？
1. 击败后永久清空，不会重复刷新；
2. 1.0全部高塔新增二阶段战斗机制，掉落古代文明核心装备图纸；
3. 未通关高塔永久存在，可反复挑战刷素材。

### 2. 世界树三层守卫Boss难度梯度？
1层：开荒可单挑，产出基础辉石；
2层：需要成型战斗帕鲁队伍，产出大量树晶矿；
3层：终局高难Boss，概率掉落突变蛋、传奇被动植入体。

### 3. 野外Alpha首领（巨型发光帕鲁）刷新规则？
每游戏日固定刷新点位，捕捉必带金色优质被动词条，可用于配种提纯极品天赋；击杀仅掉落基础素材，捕捉收益远高于击杀。

### 4. 副本能组队刷吗？
单人/多人服务器通用，最多4人同进一个地下城；多人模式Boss血量、伤害同步上调，掉落数量翻倍。

## 四、1.0配种繁育系统（改动最大，玩家提问最多）
### 1. 1.0配种底层机制大改，旧版公式全部失效？
是，新增**繁殖力数值**规则：每只帕鲁有固定繁殖力数字，数值越低越稀有强力；子代繁殖力=(父数值+母数值)÷2，**无法繁育比父母更稀有的帕鲁**，低级帕鲁不能凭空配出大龙、传说宠。

### 2. 同种类、不同种类配种区别？
1. **同种互配**：100%生出本种类帕鲁，专门提纯4金色完美被动、高个体值IV；
2. **异种互配**：按繁殖力平均值匹配对应帕鲁，部分稀有帕鲁有固定专属配方（例：百合女王=叶胖达+花丽娜）；
3. 传说帕鲁大多只能同种繁育，无跨物种配方。

### 3. 突变蛋是什么？有什么优势？
配种低概率产出突变蛋，孵化突变帕鲁：
1. 天生Alpha巨型体型，基础血量、攻击大幅提升；
2. 自带部分浓缩升星进度，省去大量捕捉；
3. 刷新专属金色/彩色被动，常规繁育无法获取；
4. 有概率变更子代物种，出高阶稀有帕鲁。

### 4. 四种繁育蛋糕分别作用？
1. 蔬菜蛋糕：基础产蛋，日常批量繁育；
2. 蘑菇蛋糕：提升个体值IV遗传概率，提纯高潜能帕鲁；
3. 突变蛋糕：大幅提高突变蛋产出概率；
4. 传奇蛋糕：提升金色稀有被动词条遗传成功率（搭配蝠乐巴特Legend被动效果翻倍）。

### 5. 蝠乐巴特为什么是配种神宠？
自带**Legend（传奇）金色被动**，大幅拉高子代继承稀有词条概率，完美毕业配种流程：高IV帕鲁×蝠乐巴特 → 子代携带Legend → 再用子代繁育战斗主力帕鲁。

### 6. 帕鲁最多能继承几个被动词条？
上限4个，彩色＞金色＞白色＞红色debuff词条；父母携带词条越少，目标词条遗传概率越高，配种前先洗干净无用负面词条。

### 7. 个体值IV怎么查看？
角色34级解锁**能力眼镜**，装备后可直接查看帕鲁生命/攻击/防御隐藏成长数值，筛选高潜力繁育亲本。

## 五、基地基建、自动化流水线FAQ
### 1. 1.0古代文明一体机有什么优势？
1. 古代物质生成器：替代伐木场+采矿场，占地极小，无限产出基础矿石、木材；
2. 古代栽培器：替换全部菜园，产量翻倍，无需手动播种收割；
3. 多功能工作台：整合帕鲁球、武器、防具、药品、材料加工，单台替代5种旧工作台。

### 2. 帕鲁工作适应性升星规则？
1~4星固定加点逻辑：
1星：最高工种适应性+1；2星：第二高工种+1；3星：第三高工种+1；4星满星：全部已有工种统一+1；
满星专职打工帕鲁效率提升70%以上，混岗全能帕鲁收益大幅削弱。

### 3. 箱子和冰箱怎么优化存储空间？
废弃冰箱，双层堆叠普通箱子：单格容量翻倍，无需安排制冷帕鲁看守，节省人力；搭配箱子自动分拣过滤器，原料自动入输入箱、成品自动存入输出箱。

### 4. 帕鲁Sanity（理智值）怎么不掉？
1. 基地全域放置饲料箱，24小时自动喂食；
2. 雪山/火山区域增设加温、降温篝火；
3. 建造帕鲁休息床，夜间自动恢复理智；
4. 营养剂道具：短时间饱腹、理智不衰减，批量流水线必备。

### 5. 1.0无限材料复制BUG怎么操作？
木柱边界漏洞：在基地边缘摆放木柱触发画面震动边界冲突，生产设施会无限返还消耗的稀有材料（世界树圣水、AI核心、烈阳钢锭）；官方正在热修复，单机慎用，联机极易炸档、清空仓库。

## 六、联机、多人服务器热门问题
### 1. 1.0支持跨平台联机吗？
PC、主机、移动端云玩互通；专用服务器内置语音，支持Discord账号绑定，单服上限32人，普通邀请码房间最多4人。

### 2. 怎么创建多人房间、分享邀请码？
1. 新建世界时开启多人模式；已有存档在「更改世界设定」打开多人开关；
2. ESC菜单顶部自动生成邀请码，好友选择「通过邀请码加入」即可；
3. 可设置密码、关闭PVP、限制建造权限。

### 3. 联机掉线、连接超时解决办法？
1. 关闭后台占用带宽软件，开启游戏加速器；
2. 房主降低画面渲染、基地帕鲁数量，减少服务器负载；
3. 专用服务器开放端口转发，关闭防火墙拦截。

### 4. 公会系统新增了什么功能？
职务权限细分、公会专属标记、成员登录通知、公会仓库共享物资、集体配种牧场，多人团队基建协作更方便。

### 5. 联机PVP规则？
默认关闭，世界设定手动开启；仅玩家、己方帕鲁可互相造成伤害，基地建筑不受PVP破坏；禁止抢夺他人终端内帕鲁。

## 七、闪退、卡顿、BUG、硬件优化FAQ
### 1. 游戏启动黑屏、UE虚幻引擎崩溃报错？
1. 卸载全部旧版MOD，MOD与1.0引擎架构完全不兼容；
2. 更新NVIDIA/AMD显卡最新驱动；
3. Steam验证游戏文件完整性，补全C++运行库；
4. 调低画面渲染距离、帕鲁模型显示数量，释放显存。

### 2. 频繁闪退、内存溢出怎么修复？
1. 设置虚拟内存：16G内存初始24000MB、最大32000MB，重启电脑；
2. 缩小基地规模，减少同时加载的帕鲁、建筑；
3. 关闭全局光照、体积云等高耗显存特效；
4. 单基地不要放置超过60只打工帕鲁，避免CPU计算过载。

### 3. 存档手动备份路径在哪？
Steam版：`C:\Users\用户名\AppData\Local\Pal\Saved\SaveGames`
每次大更新前复制文件夹备份，防止炸档丢失进度。

### 4. 低配电脑能流畅运行1.0吗？
最低8G内存+GTX1050，全部特效拉低、关闭动态阴影、限制帕鲁同屏数量；16G内存+3060可中高画质稳定60帧。

## 十、全网高频误区&冷门机制FAQ
### 1. 传说帕鲁能野外捕捉吗？
世界树秘境三层、禁猎异变区极低概率刷新野生传说；常规地图无野生传说，大多只能配种繁育。

### 2. 死亡惩罚改了什么？
新增「紧急脱离」替代强制死亡，使用后不掉落任何物资；正常死亡仅掉落少量背包物资，帕鲁不会丢失。

### 3. 野生帕鲁词条颜色等级划分？
彩色（最强增益）＞金色＞白色＞红色（负面debuff，如贪吃、懒惰）；打工帕鲁优先彩色「卓越技艺」，战斗帕鲁堆金色传奇、猛攻词条。

### 4. 帕鲁 sanity 归零会逃跑吗？
1.0修改机制：不会逃跑，但工作效率降低90%，停止一切生产，长期低理智会持续掉生命值直至倒地。

### 5. 武器装备能给帕鲁穿戴吗？
仅饰品、指挥棒类道具可装备给帕鲁；武器、防具仅玩家角色使用，无法给帕鲁佩戴。

### 6. 废弃旧基地会占用游戏内存吗？
会，推荐废弃基地全部拆除，残留建筑、帕鲁AI持续后台计算，导致长期卡顿闪退。

### 7. 1.0会推出付费DLC、逼氪内容吗？
官方确认主线全部免费，后续更新为免费新帕鲁、限时活动、创意工坊；无付费剧情、付费强力帕鲁、付费数值道具，仅外观装饰类可选付费。

### 8. 能不能删除不需要的被动词条？
无直接删除功能，只能配种提纯：亲本仅保留目标优质词条，洗掉负面、无用词条，逐步迭代完美个体。

### 9. 变异帕鲁和普通Alpha首领区别？
普通Alpha仅体型大、基础属性小幅提升；突变帕鲁额外带专属被动、浓缩进度、可刷新稀有物种，养成上限更高。

### 10. 配种牧场需要几只帕鲁才能产蛋？
必须1公1母同放牧场，投放对应蛋糕自动开始繁育；单只帕鲁无法产蛋，无无性繁殖机制。
