# 补充：全热门帕鲁专项FAQ（战斗/打工/传说/突变/配种/捕捉/机制高频问题，可直接入库RAG）
## 一、传说顶级战斗帕鲁高频问答（腾炎龙、空涡龙、唤冬兽、魔渊龙、拔刀鱼、云海巨鲸）
### 1. 空涡龙为什么是1.0综合天花板？野外在哪抓？
- 核心优势：**采集8级T0打工+全游戏最快飞行坐骑+大范围AOE导弹轰炸**，打工战斗两用；传说种族值，单刷世界树三层Boss无压力。
- 捕捉点位：天阳乡浮空岛熔岩池 `-552, -1333`，68级Alpha首领定点刷新，游戏单日重置一次。
- 常见疑问：能不能配种出空涡龙？
  可以，空涡龙仅能**同种繁育**，无法跨物种杂交产出；突变蛋有极低概率随机刷出空涡龙。
### 2. 腾炎龙捕捉位置、配种公式、开荒价值？
- 捕捉：天坠之地南岸50~60级野外可直接抓捕，无需终局地图。
- 打工定位：生火工种平民T0，3星生火直接满级10级，替代难获取的燎火舞伶。
- 配种链核心枢纽：腾炎龙+焰煌=燎火舞伶；腾炎龙+灵曦龙=墨罗娜，绝大多数高阶打工帕鲁繁育都绕不开它。
- 战斗：大范围火焰AOE，单刷高塔、副本效率拉满。
### 3. 唤冬兽（冰传说）有什么独有功能？在哪刷新？
- 专属能力：骑乘后玩家普攻全部转为冰属性，大幅提升冰伤；基地自带**顶级制冷光环**，雪山基地不用额外制冷帕鲁。
- 刷新点位：地图最北部冰封高山，75级固定Alpha首领，每日游戏时间刷新。
- 常见疑问：唤冬兽能和腾炎龙配种吗？
  两者繁殖力差距极大，子代只会产出中等稀有帕鲁，无法出双传说后代，不推荐配对。
### 4. 云海巨鲸特殊机制问答
1. 为什么别人能进世界树深层，我飞不进去？
   只有云海巨鲸能无障碍穿越世界树顶层空气屏障，其他飞行帕鲁会被地形拦截。
2. 巨鲸能打工吗？
   无任何工作适应性词条，纯代步、多人飞行坐骑，不能参与基地生产。
3. 野外能抓到巨鲸吗？
   仅世界树秘境三层Boss概率掉落巨鲸蛋，无野生个体，只能孵化/配种获得。
### 5. 拔刀鱼（天阳乡传说）核心疑问
1. 附体近战强度如何？
   全游戏单挑天花板，人帕一体切换太刀，空战、Boss战无敌，克制所有龙系Boss。
2. 捕捉难度高吗？
   天阳乡浮空岛高空刷新，血量厚，必须麻痹+传奇球捕捉，建议带唤冬兽冰系控制。
3. 能不能配种提纯词条？
   同种繁育，突变蛋有概率刷彩色专属近战词条。
### 6. 魔渊龙优缺点、获取方式
- 优势：暗系大范围持续腐蚀AOE，群体清小怪、多人副本输出T0；自带高防御坦度。
- 短板：无任何打工词条，纯战斗宠，占地大、基地不适合长期存放。
- 获取：世界树三层终局Boss掉落蛋，野外无刷新。

## 二、T0打工专用帕鲁全网高频问题（墨罗娜、灵曦龙、覆海龙、空涡龙、燎火舞伶、电汪汪）
### 1. 墨罗娜为什么是手工天花板？
- 原生自带**手工8级**，体型小巧不占空间，不需要睡觉休息，自动化武器、帕鲁球、药品生产线必备，1.0新增世界树左下角树林定点Alpha可抓。
- 常见误区：墨罗娜能采矿/伐木吗？
  原生无采矿伐木词条，仅手工专精，不要分配采矿工位，效率极低。
### 2. 灵曦龙（搬运T0）相关疑问
1. 灵曦龙在哪抓？天阳乡69级Alpha首领固定点位。
2. 搬运效率有多强？原生搬运6，4星直接满级10级，全游戏专职搬运第一，解决生产线物资堆积卡死问题。
3. 配种简化路线：腾炎龙+墨罗娜直接繁育灵曦龙，中期就能批量培育。
### 3. 覆海龙 vs 霄龙（浇水工种怎么选）
- 霄龙：浇水理论天花板（原生浇水8，2星满级），但仅世界树70+区域刷新，无法批量捕捉，只能配种。
- 覆海龙（平民首选）：原生浇水7，野外55级Alpha可定点抓捕，3星浇水拉满，大规模农田自动化性价比碾压霄龙。
### 4. 电汪汪（波鲁杰克斯）发电疑问
1. 1.0发电唯一T0，世界树毒雾区刷新，原生发电8级，全程不间断供电，不用频繁喂食。
2. 能兼职其他工位吗？
   仅发电单词条，混岗会大幅降低整体工作速度，发电站单独放置最佳。
### 5. 燎火舞伶获取难，有没有平替？
- 平替腾炎龙，3星生火满级，野外可抓；燎火舞伶仅火山夜晚低概率刷新，繁育公式：腾炎龙+焰煌。
### 6. 企丸丸、捣蛋猫开荒打工疑问
1. 企丸丸：浇水+手工双词条，新手种田、烹饪过渡神宠，第一禁猎区20级可抓。
2. 捣蛋猫：采集/手工/搬运三杂词条，前期物资紧缺过渡用，中期全部替换单词条专精帕鲁。
### 7. 1.0打工机制改动：多词条全能帕鲁废了？
是的，升星规则重做：1星仅提升最高工种，4星才统一小幅加成全部词条；多杂词条帕鲁升星收益极低，**单词条专精帕鲁全面崛起**，混岗全能宠彻底淘汰。

## 三、配种、突变蛋、词条遗传专属帕鲁问题（蝠乐巴特、突变帕鲁、Alpha首领）
### 1. 蝠乐巴特为什么是配种必备神宠？
自带金色【传奇Legend】被动，大幅提升子代继承稀有金色/彩色词条概率；标准毕业流程：高IV亲本 × 蝠乐巴特，子代携带Legend后再繁育战斗/打工主力，大幅减少洗词条周期。
### 2. 突变蛋相关全热门疑问
1. 突变蛋孵化出的帕鲁和普通Alpha首领区别？
   突变帕鲁=永久巨型Alpha体型，**天生2星浓缩进度、90~100满IV个体值、必带2个彩色专属词条**，常规配种无法刷出彩色突变被动，部分突变会直接跨物种产出高阶稀有帕鲁。
2. 怎么提高突变蛋产出概率？
   投放**突变蛋糕**，基础概率从5%提升至22%；固定一对帕鲁持续批量繁育，更容易出突变蛋。
3. 突变蛋能刷出传说帕鲁吗？
   极低概率可以随机刷出空涡龙、拔刀鱼等传说，比野外捕捉效率高很多。
4. 突变专属彩色词条有哪些强力款？
   不朽（持续回血）、护崽（繁育提速）、重甲（爆炸伤害免疫）、独行飞行（坐骑免灼烧中毒）。
### 3. Alpha首领帕鲁常见误区
1. 野外Alpha首领捕捉有什么优势？
   必带至少1条金色优质被动，基础属性比普通个体高20%，是配种优质亲本。
2. Alpha首领能繁殖突变蛋吗？
   可以，两只Alpha配对，突变蛋产出概率额外小幅提升。
### 4. 帕鲁词条遗传核心疑问
1. 最多能遗传几条被动？上限4条，彩色＞金色＞白色＞红色负面词条。
2. 怎么洗掉懒惰、贪吃等负面词条？
   配种时亲本只保留优质词条，不带负面词条，子代不会继承缺陷词条；无单独删除词条道具。
3. 个体值IV怎么看？角色34级解锁能力眼镜，可查看生命/攻击/防御隐藏成长数值，高IV帕鲁繁育后代成长上限更高。
